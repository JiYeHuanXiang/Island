常用命令:

  • .r - 掷骰子 (例如: .r 3d6)
  • .coc - COC角色生成
  • .dnd - DND角色生成
  • .help - 显示帮助信息

连接方式说明:

  • WebSocket - 直接连接到QQ机器人的WebSocket服务
  • HTTP API - 通过HTTP API与QQ机器人通信
  • 反向WebSocket - QQ机器人连接到本程序
